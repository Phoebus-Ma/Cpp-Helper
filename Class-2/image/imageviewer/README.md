
# Introduction

This a image viewer, support image format:

- BMP  (.bmp, .dib)
- Gif  (.gif)
- JPEG (.jpeg, .jpg, .jpe)
- PNG  (.png)
- SVG  (.svg, .svgz)
- TIFF (.tif, .tiff)
- ICO  (.ico)
- WBMP (.wbmp)
- WebP (.webp)
- X    (.icns)
- PBM  (.pbm)
- PGM  (.pgm)
- PPM  (.ppm)
- TGA  (.tga, .icb, tpic, .vda, .vst)
- XBM  (.xbm)
- XPM  (.xpm)

The example based on qt-5.15.


# Usage

Command line:

```bash
$ imageviewer sample.png
```

UI:

```bash
$ imageviewer
```
